## Principais Funcionalidades da Automação:
- Buscar base de dados no sistema
- Ler a base de dados
- Validar a base de dados com CPF's
- Criar um novo arquivo com a validação, retornando "Válido" ou "Inválido"

- Mandar e-mail
- Bot Whatsapp/Telegram
